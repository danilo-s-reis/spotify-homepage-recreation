## Sobre o projeto...  
Recriação estática da antiga interface da homepage do Spotify, desenvolvido para fins de estudo.

## Stack  
HTML + CSS + JS

## Disclaimer
Ao tentar acessar o site, o browser pode acabar atestando um falso positivo de site fraudulento e bloquear a conexão, entretanto o objetivo desse projeto é meramente de estudo sobre front-end e não possui nenhum tipo de coleta de dados, podendo esse fato ser atestado facilmente pelo conteúdo presente neste repositório.
